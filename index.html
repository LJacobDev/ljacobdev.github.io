Demo Library App with Staff Dashboard and AI Chat Agent (Vue 3 / Nuxt 4)

An important note about this Library app is that it was performed as an exercise in using LLMs to make a working site in the shortest time possible, so it emphasized shipping functionality quickly using timesaving measures such as only using manual testing. Starting from absolute zero, it had tech stacks researched for fit and specifications made in 3 days, and it was built with manual testing in 3 more days. There is a plan for testing coverage and other quality improvements to be done on this app, and its current state of being made in a rush was both intentional and not a reflection of the kind of Test Driven Development that I prefer to build my code by.
